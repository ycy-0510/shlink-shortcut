Shlink iOS Shortcut

Generate self-hosted Shlink short URLs directly from your iPhone with a single tap.

Get Shortcut Setup Guide

1 What it does

This shortcut connects to your self-hosted Shlink instance via API to create short URLs from your clipboard or share sheet instantly.

2 Setup Instructions

When prompted during the first run, please provide:

  • Endpoint URL:

    The full URL of your Shlink instance (e.g., https://sh.link.com)

  • API Key:

    Your unique API key generated in the Shlink dashboard.

Quick Reference

Need help setting up Shlink? Refer to the official documentation for Docker installation and API management.
